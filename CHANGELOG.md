        ## [1.0.0] - 2026-05-01

        ### Added
        - Initial release of FocusMon
        - - Pomodoro-style focus sessions with completion and give-up outcomes
          - - XP, levels, focus coins, streaks, and total focus-hour tracking
            - - Monster companion with level-based evolution art
              - - Quest, journal, profile, creature, vault, and leaderboard screens
                - - Offline persistence with AsyncStorage
                  - - Optional Supabase Auth and Postgres sync
                    - - Animated timer, XP, and level-up moments with Reanimated
                      - - Built with React Native + Expo 56 + TypeScript
                        - - NativeWind + Tailwind CSS for styling
                          - - Zustand state stores for user, creature, and session data
